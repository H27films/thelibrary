## Packages
framer-motion | Essential for the fluid, high-end editorial hover and scroll animations
react-hook-form | Form state management for the minimalist item entry overlay
@hookform/resolvers | Zod validation integration for forms
clsx | Utility for conditional class merging
tailwind-merge | Utility for conditional class merging

## Notes
The application features a single-page fixed-viewport split-pane architecture.
Animations use Framer Motion to create the "Digital Editorial" aesthetic.
Custom typography (Cormorant Garamond) is loaded via Google Fonts in index.css.
